# ShopNova - Next.js E-commerce (No Database)

Modern e-commerce frontend built with **Next.js 14 App Router**, **TypeScript**, **TailwindCSS**, and reusable UI components inspired by shadcn/ui.

## Features
- Home, Shop, Product Details, Cart, Checkout, Orders, and Success pages
- 30+ static products from local JSON files
- Cart, wishlist, and orders persisted with LocalStorage
- Filtering, sorting, and search on shop page
- Dark mode toggle, toasts, skeleton loading, responsive layout

## Run locally
```bash
npm install
npm run dev
```
Then open `http://localhost:3000`.
